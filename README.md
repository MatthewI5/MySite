# MySite

## API

The API can be run separately to provide extra functionality (links to Immich and Ghost).

This requires setup of an .env file and a Python environment. 


```
IMMICH_BASE = "[YOUR IMMICH SERVER BASE URL]"
GHOST_BASE = "[YOUR GHOST ADMIN BASE URL]"
GHOST_KEY = "[GHOST API KEY]"
IMMICH_ALBUM = "[IMMICH ALBUM ID]"
```
